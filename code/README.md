# Code

This directory contains all code, scripts, and notebooks for data processing and analysis.

## Contents

* **Data processing scripts**: Python scripts for loading, sampling, and transforming the Yelp dataset
* **PostgreSQL scripts**: SQL table creation, data loading, and query execution
* **Spark scripts**: PySpark code for distributed data processing
* **Jupyter notebooks**: Interactive analysis and visualization notebooks
* **Performance benchmarking**: Scripts to measure and compare query execution times

## Key Files

* Data extraction and sampling from JSON files
* Database schema creation and initialization
* ETL pipeline for PostgreSQL
* Spark DataFrame operations
* Query performance measurement utilities

## Dependencies

See requirements or setup instructions in the main README for installing necessary libraries (pandas, psycopg2, pyspark, etc.)

# Queries

This directory contains SQL queries and Spark operations used in the project.

## Organization

* `postgres/` - PostgreSQL SQL queries
* `spark/` - PySpark or Scala code for Spark operations

## Query Categories

* **Aggregation queries**: COUNT, SUM, AVG operations on reviews and businesses
* **Join queries**: Multi-table queries combining users, businesses, and reviews
* **Analytical queries**: Complex queries for business insights and user behavior analysis
* **Performance benchmarks**: Identical queries run on both PostgreSQL and Spark for comparison

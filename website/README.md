# PostgreSQL vs Apache Spark - Portfolio Website

Modern, interactive portfolio website showcasing the performance comparison between PostgreSQL and Apache Spark on the Yelp Open Dataset.

## Features

- **Modern Design**: Dark theme with gradient animations and smooth scrolling
- **Interactive Components**: Expandable code blocks, query selector, and performance charts
- **Performance Visualizations**: Interactive Recharts comparing execution times across different data sizes
- **Syntax Highlighting**: Beautiful code presentation for SQL and PySpark
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile
- **Animations**: Powered by Framer Motion for smooth, professional transitions

## Tech Stack

- **React 18** - Component-based UI
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **Recharts** - Performance visualization charts
- **Framer Motion** - Smooth animations
- **React Syntax Highlighter** - Code syntax highlighting
- **React Icons** - Professional icon library

## Getting Started

### Installation

\`\`\`bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
\`\`\`

## Deployment

Deploys automatically to GitHub Pages via GitHub Actions on push to main branch.

Site will be available at: \`https://[username].github.io/postgres-spark/\`
